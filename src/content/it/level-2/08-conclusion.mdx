# Il Tocco Finale: Visual Mode.

Prima di chiudere il Livello 2, dobbiamo parlare dell'ultima modalità essenziale: il **Visual Mode**. Mentre in Normal Mode operi alla cieca (affidandoti alla grammatica), in Visual Mode evidenzi il testo prima di agire.

<Step number="1" title="Selezionare il Testo">
Entra in Visual Mode e usa i movimenti standard (`h,j,k,l,w,b,e,$`) per evidenziare blocchi di testo:
</Step>

  data={[
    { key: "v", label: "Visual (Carattere)", desc: "Seleziona carattere per carattere. Ottimo per porzioni di riga." },

/>

<Step number="2" title="Agire sulla Selezione">
Una volta evidenziato il testo, premi un Operatore (`d, c, y`) o usa questi comandi speciali per la selezione:
</Step>

<ShortcutGrid 
  data={[
    { key: ">", label: "Indenta (Avanti)", desc: "Sposta il testo selezionato verso destra." },

<Card title="⚡ Il Segreto di Ctrl+V (Visual Block)">
  Seleziona più righe con `Ctrl+v` (un rettangolo verticale). Premi `I` (Insert maiuscolo), scrivi qualcosa (es: `// ` per commentare) e premi `ESC`. Il testo apparirà magicamente su *tutte* le righe selezionate!
</Card>
