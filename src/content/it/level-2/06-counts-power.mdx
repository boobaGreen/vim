# Moltiplicare l'Azione.

Vim non è solo preciso; è scalabile. Inserendo un numero (**Conteggio**) prima di un comando, puoi ripeterlo quante volte vuoi senza fatica.

### La Formula Completa
Ricordi la grammatica? `[VERBO] + [CONTEGGIO] + [MOVIMENTO]`

- `3w`: Salta avanti di 3 parole.
- `d2w`: Cancella le prossime 2 parole.
- `c3e`: Cambia fino alla fine della terza parola successiva.

### Quando usare i conteggi?
I conteggi sono perfetti quando vedi una ripetizione ovvia. Invece di premere `dw` tre volte, premi `d3w`. Risparmi tasti, risparmi tempo, riduci l'attrito cognitivo.
