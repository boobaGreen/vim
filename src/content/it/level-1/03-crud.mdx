# Creare e Distruggere.

In Vim, non si scrive solo. Si manipola la materia testuale con gesti precisi.

<InfoBox type="tip" title="IL MANTRA DI OGGI">
Dimentica le frecce. Inizia a pensare per azioni. Ogni tasto è un comando di trasformazione.
</InfoBox>

</Step>

</InfoBox>

<Card title="💡 Il Segreto del Punto (.)">
  Il tasto `.` (punto) ripete l'ultimo comando di modifica. Esempio: se fai `dd` per cancellare una riga, poi premi `.` per cancellarne un'altra. È il comando più sottovalutato di Vim!
</Card>

<Card title="⚡ Esercizio Pratico">
  Nel terminale, prova questa sequenza:
  1. Premi `i` per entrare in Insert Mode
  2. Scrivi "Vim è potente"
  3. Premi `ESC` per tornare in Normal Mode
  4. Premi `dd` per cancellare la riga
  5. Premi `u` per annullare (Undo)
  6. Premi `.` per ripetere l'ultimo comando (ri-cancella!)
</Card>
