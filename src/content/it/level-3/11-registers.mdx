# Oltre il Copia-Incolla.

In un editor normale, hai un solo "appunti" (clipboard). In Vim, ne hai dozzine. Questi si chiamano **Registri**.

### Tipi di Registri

  data={[

### Come usare un registro
La formula è: **`"` + [NOME_REGISTRO] + [AZIONE]**

<ShortcutGrid 
  data={[
    { key: '"ayw', label: 'Copia in Registro', desc: 'Copia una parola specificatamente nel registro "a".' },
    { key: '"ap', label: 'Incolla da Registro', desc: 'Incolla il contenuto del registro "a".' },
    { key: '"+y', label: 'Copia al Sistema', desc: 'Copia la selezione negli appunti globali.' },
  ]}
/>

### Perché è utile?
Puoi copiare un pezzo di codice nel registro `f` (per 'funzione') e un altro nel registro `v` (per 'variabile'), e incollarli quando vuoi senza che uno sovrascriva l'altro.
