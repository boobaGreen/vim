# The Final Touch: Visual Mode.

Before closing Level 2, we must discuss the last essential mode: **Visual Mode**. While in Normal Mode you operate blindly (relying on grammar), in Visual Mode you highlight text before acting.

<Step number="1" title="Selecting Text">
Enter Visual Mode and use standard motions (`h,j,k,l,w,b,e,$`) to highlight blocks of text:
</Step>

<ShortcutGrid 
  data={[
    { key: "v", label: "Visual (Character)", desc: "Selects character by character. Great for parts of a line." },
    { key: "V", label: "Visual Line", desc: "Selects entire lines. Perfect for moving code blocks." },
    { key: "Ctrl+v", label: "Visual Block", desc: "Selects rectangular blocks. Pure magic for column editing." },
  ]}
/>

<Step number="2" title="Acting on Selection">
Once the text is highlighted, press an Operator (`d, c, y`) or use these special commands for the selection:
</Step>

<ShortcutGrid 
  data={[
    { key: ">", label: "Indent (Right)", desc: "Moves the selected text to the right." },
    { key: "<", label: "De-indent (Left)", desc: "Moves the selected text to the left." },
    { key: "u / U", label: "Case Toggle", desc: "Forces lowercase (`u`) or UPPERCASE (`U`) on the selection." },
  ]}
/>

<Card title="⚡ The Secret of Ctrl+V (Visual Block)">
  Select multiple lines with `Ctrl+v` (a vertical rectangle). Press `I` (Capital Insert), type something (e.g., `// ` to comment) and press `ESC`. The text will magically appear on *all* selected lines!
</Card>
