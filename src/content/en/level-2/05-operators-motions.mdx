# The Dance of Operators.

Welcome to Level 2. Now that you know the basics, it's time to learn how to "speak" fluently. In Vim, an **Operator** defines *what* to do, and a **Motion** defines *what to do it on*.

<Step number="1" title="The Big Three Operators">
Let's learn the main verbs:
</Step>

</Step>

<ShortcutGrid

    { key: "b", label: "Back", desc: "Jumps to the start of the previous word." },
  ]}
/>

<Step number="3" title="The Magic Combination">
Try combining an operator with a motion in the terminal:
</Step>

- `dw`: Deletes until the next word.
- `cw`: Changes the current word (deletes and writes).
- `yw`: Copies the word.

<Card title="Quick Challenge">
  If you type `de`, what do you think will happen? Try it in the terminal! It will delete until the end of the current word.
