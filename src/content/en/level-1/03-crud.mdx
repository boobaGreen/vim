# Create and Destroy.

In Vim, you don't just write. You manipulate textual matter with precise gestures.

<InfoBox type="tip" title="TODAY'S MANTRA">

</InfoBox>

</Step>

<InfoBox type="warning" title="IMPORTANT">
To return to **Normal Mode** after using any insert command, always press <Kbd>ESC</Kbd>. Without this key, you'll keep typing instead of commanding.
</InfoBox>

<Card title="💡 The Secret of the Dot (.)">
  The `.` (dot) key repeats the last edit command. Example: if you type `dd` to delete a line, then press `.` to delete another one. It's the most underrated Vim command!
</Card>

<Card title="⚡ Practical Exercise">
  In the terminal, try this sequence:
  1. Press `i` to enter Insert Mode
  2. Type "Vim is powerful"
  3. Press `ESC` to return to Normal Mode
  4. Press `dd` to delete the line
  5. Press `u` to undo
  6. Press `.` to repeat the last command (re-deletes!)
</Card>
